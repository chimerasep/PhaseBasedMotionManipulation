About the Project

Our project implements phase-based motion manipulation in Unity to reduce cybersickness in VR environments. Using Fast Fourier Transform (FFT) and real-time frame processing, we manipulate motion intensity while maintaining visual quality.

How It Works

Frame Processing

Our system captures frames from Unity, converts them to YIQ color space, and processes the Y channel using FFT and Complex Steerable Pyramid decomposition for motion manipulation.

FFT & Pyramid

Custom compute shader implementation of Fast Fourier Transform and Complex Steerable Pyramid for efficient real-time processing of motion data.

Motion Control

Phase-based manipulation of motion using complex number operations, butterfly algorithms, and steerable pyramid decomposition for precise control.

Technical Implementation

Our solution uses a custom MotionMagnificationProcessor in Unity that:

  • Captures and processes frames in real-time using compute shaders
  • Implements 2D FFT using butterfly operations for efficient processing
  • Utilizes Complex Steerable Pyramid decomposition for multi-scale motion analysis
  • Manipulates motion through phase difference analysis at multiple scales
  • Applies anti-aliasing and windowing functions to reduce artifacts
  • Processes frames in YIQ color space for better motion control

The system maintains high performance through:

  • GPU-accelerated compute shader operations
  • Optimized buffer management for real-time processing
  • Efficient memory usage with structured buffers
  • Parallel processing of frame data
  • Multi-scale motion analysis using Complex Steerable Pyramid
